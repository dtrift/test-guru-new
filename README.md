# Приложение test-guru

Список сущностей:

1. Пользователь

2. Тест

3. Вопрос

4. Ответ

5. Результат теста

Основные возможности:

* Создание и управление тестами

* Аутентификация пользователей

* Базовое разграничение прав доступа

* Выполнение тестов

* Публикация результатов в социальных сетях
